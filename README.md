https://pkg.go.dev/plugin

Inspirado em
https://github.com/vladimirvivien/go-plugin-example/tree/master

Como fazer o build e testar

Build do main
build.sh

Em cada diretorio lib1 ... lib3
cd lib1
./build.sh

cd lib2
./build.sh

cd lib3
./build.sh

Depois é só rodar o main
./main
